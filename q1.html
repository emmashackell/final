http://localhost:3002/send/

This action URL will not work on GitHub Pages. GitHub Pages is a static site hosting service, which means that it does not support running a backend server or processing POST requests. The URL `http://localhost:3002/send/` refers to a local server running on your own machine, which is only accessible locally. Since GitHub Pages cannot run local servers, this action URL won't work when hosted on GitHub Pages.
